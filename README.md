# phutball

Philosopher's Football


### Quick setup instructions:

 1. `npm install`
 2. `npm run compile`
 3. `npm start`
 4. Then, simply navigate to `http://localhost:8080` in your web browser.

### Instructions:

Rules and helpful tips are available in-game!
